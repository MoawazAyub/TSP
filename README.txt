TSP:
TSP is a simple solution to traveling sales person problem using genetic algorithm.
How to get started:
Load the .sln file in visual studio to run the project. After running the project you will be asked to enter the population that you want to select. The Crossover Technique and the Mutation rate. You can enter 1, 2, 3 as input. However for population and Mutation rate you can enter any number and You also have to enter desired fitness.
The output will be Stored in Output.txt that can be used to make the graph using “mat.m” script in MATLAB.
The dataset for this project has been downloaded from http://people.sc.fsu.edu/~jburkardt/datasets/cities/cities.html
